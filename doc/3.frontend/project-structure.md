# 前端项目结构

## 概述

前端项目基于 Vue 3 + TypeScript + Vite 构建，采用组合式 API 和 Pinia 状态管理，使用 Tailwind CSS 进行样式开发。

## 目录结构

```
frontend/
├── src/
│   ├── App.vue              # 根组件
│   ├── main.ts              # 入口文件
│   ├── vite-env.d.ts        # Vite 类型声明
│   │
│   ├── components/          # Vue 组件
│   │   ├── DanmakuTestPanel.vue
│   │   ├── NotificationToast.vue
│   │   ├── PlayUnlockModal.vue
│   │   ├── SessdataWarningModal.vue
│   │   ├── SettingsPanel.vue
│   │   │
│   │   ├── danmaku/         # 弹幕相关
│   │   ├── infobar/         # 信息栏
│   │   ├── layout/          # 布局组件
│   │   ├── mainview/        # 主视图
│   │   ├── music/           # 音乐播放
│   │   └── panels/          # 面板组件
│   │
│   ├── composables/         # 组合式 API
│   ├── stores/              # Pinia 状态管理
│   ├── types/               # TypeScript 类型
│   └── utils/               # 工具函数
│
├── index.html               # HTML 入口
├── package.json             # 依赖配置
├── vite.config.ts           # Vite 配置
├── tailwind.config.js       # Tailwind 配置
└── tsconfig.json            # TypeScript 配置
```

## 技术栈

| 技术 | 版本 | 用途 |
|-----|------|------|
| Vue 3 | 3.x | 前端框架 |
| TypeScript | 5.x | 类型系统 |
| Vite | 5.x | 构建工具 |
| Pinia | - | 状态管理 |
| Tailwind CSS | 3.x | 样式框架 |

## 开发命令

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
