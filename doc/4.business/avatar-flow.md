# 虚拟形象渲染流程

## 概述

虚拟形象渲染模块基于 EasyVtuber 实现，通过 ONNX Runtime 在 GPU 上运行 THA3/THA4 模型，实时驱动虚拟形象并推流到前端。

## 模型说明

### THA3/THA4 模型

```
┌─────────────────────────────────────────────────────────────────┐
│                    THA 模型说明                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  THA (Talking Head Animation) 是虚拟形象驱动的核心模型         │
│                                                                 │
│  输入:                                                          │
│  - 静态角色图片 (PNG, 透明背景)                                │
│  - 表情参数 (眼睛方向、嘴型开合、眉毛位置等)                   │
│                                                                 │
│  输出:                                                          │
│  - 动画帧 (PNG, 透明背景)                                      │
│                                                                 │
│  版本区别:                                                      │
│  - THA3: 基础版本，表情参数较少                                │
│  - THA4: 增强版本，更多表情参数，更自然的动画                  │
│                                                                 │
│  模型文件位置:                                                  │
│  backend/EasyVtuber/data/models/                                │
│  ├── tha3/                                                     │
│  │   └── *.onnx                                                │
│  └── tha4/                                                     │
│      └── *.onnx                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 配置参数

```yaml
# config.yaml
easyvtuber:
  enabled: true
  
  character: "lambda_00"  # 角色名称
  
  input:
    type: "debug"         # 输入类型: debug / osf / mouse
    osf_address: "127.0.0.1:11573"  # OpenSeeFace 地址
    mouse_range: "0,0,1920,1080"    # 鼠标活动范围
  
  model:
    version: "v3"         # 模型版本: v3 / v4
    precision: "half"     # 精度: half / full
    separable: true       # 是否使用分离模型
    use_tensorrt: true    # 是否使用 TensorRT
    use_eyebrow: true     # 是否启用眉毛控制
  
  performance:
    frame_rate: 30        # 目标帧率
    interpolation: "x2"   # 插帧设置
    super_resolution: "off"  # 超分辨率
    ram_cache: "2gb"      # 内存缓存大小
    vram_cache: "2gb"     # 显存缓存大小
  
  output:
    websocket:
      enabled: true
      host: "localhost"
      port: 8765
```

## 性能优化

```
┌─────────────────────────────────────────────────────────────────┐
│                    性能优化策略                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. GPU 加速                                                    │
│     - ONNX Runtime CUDA Execution Provider                     │
│     - TensorRT 加速 (可选)                                     │
│                                                                 │
│  2. 精度优化                                                    │
│     - 半精度 (FP16) 推理                                       │
│     - 减少显存占用，提升速度                                   │
│                                                                 │
│  3. 缓存策略                                                    │
│     - RAM 缓存: 缓存模型权重                                   │
│     - VRAM 缓存: 缓存中间结果                                  │
│                                                                 │
│  4. 批处理                                                      │
│     - 多帧并行推理                                             │
│     - 减少 GPU 调用次数                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
