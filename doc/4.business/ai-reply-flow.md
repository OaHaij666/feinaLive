# AI 回复流程

## 概述

AI 回复流程是系统的核心功能之一，负责接收弹幕、调用大语言模型生成回复、合成语音并推送到前端。

## 整体流程图

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                           AI 回复完整流程                                           │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │

│  │   - 是否为管理员指令 (已在上层处理)                                        │  │
│  │   - sleep 模式下是否过滤                                                   │  │
│  │                                                                             │  │
```

## LangGraph 节点详解

### 1. retrieve_rag - 知识检索

```
┌─────────────────────────────────────────────────────────────────┐
│                    retrieve_rag 节点                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 从知识库中检索相关内容                                   │
│                                                                 │
│  输入: state["text"] (用户输入)                                │
│  输出: state["rag_context"] (检索结果列表)                     │
│                                                                 │
│  当前实现: 返回空列表 (预留接口)                               │
│                                                                 │
│  未来扩展:                                                      │
│  - 向量数据库检索                                              │
│  - 语义相似度匹配                                              │
│  - 知识库问答                                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. retrieve_user_memory - 用户记忆检索

```
┌─────────────────────────────────────────────────────────────────┐
│                retrieve_user_memory 节点                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 获取用户的记忆上下文                                     │
│                                                                 │
│  输入: state["user"] (用户名)                                  │
│  输出: state["user_memory_context"] (记忆上下文字符串)         │
│                                                                 │
│  流程:                                                          │
│  1. get_user_profile(user) ──── 获取用户画像                  │
│  2. user_profile.get_memory_context() ──── 构建上下文         │
│                                                                 │
│  记忆上下文包含:                                                │
│  - 用户印象 (impression)                                       │
│  - 长期记忆 (long_term_memory)                                 │
│  - 近期对话 (recent_messages)                                  │
│  - 关键话题 (key_topics)                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. build_prompt - 构建提示词

```
┌─────────────────────────────────────────────────────────────────┐
│                    build_prompt 节点                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 构建 LLM 的输入提示词                                    │
│                                                                 │
│  输入:                                                          │
│  - state["text"] (用户输入)                                    │
│  - state["rag_context"] (知识检索结果)                         │
│  - state["user_memory_context"] (用户记忆)                     │
│                                                                 │
│  输出: state["prompt"] (消息列表)                              │
│                                                                 │
│  提示词结构:                                                    │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ System: 你是飞娜，一个活泼可爱的虚拟主播...             │   │
│  │                                                         │   │
│  │ 【相关知识】                                            │   │
│  │ {rag_context}                                          │   │
│  │                                                         │   │
│  │ 【用户信息】                                            │   │
│  │ {user_memory_context}                                  │   │
│  │                                                         │   │
│  │ User: 之前的对话历史...                                 │   │
│  │ Assistant: 之前的回复...                                │   │
│  │ User: {当前弹幕内容}                                    │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. stream_llm - LLM 生成

```
┌─────────────────────────────────────────────────────────────────┐
│                    stream_llm 节点                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 调用 LLM 生成回复                                        │
│                                                                 │
│  输入: state["prompt"] (提示词)                                │
│  输出: state["response"] (生成的回复)                          │
│                                                                 │
│  配置参数:                                                      │
│  - model: 模型名称                                             │
│  - temperature: 温度参数 (0.7)                                 │
│  - top_p: Top-P 参数 (0.9)                                    │
│  - max_tokens: 最大 Token 数 (200)                            │
│                                                                 │
│  流式输出:                                                      │
│  async for chunk in ai.chat_stream(request):                  │
│      chunks.append(chunk)                                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 5. stream_tts - 语音合成

```
┌─────────────────────────────────────────────────────────────────┐
│                    stream_tts 节点                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 将文本转换为语音                                         │
│                                                                 │
│  输入: state["response"] (回复文本)                            │
│  输出: state["audio_data"] (音频数据)                          │
│                                                                 │
│  TTS 服务: 火山引擎声音复刻                                     │
│                                                                 │
│  配置参数:                                                      │
│  - appid: 应用ID                                               │
│  - access_token: 访问令牌                                      │
│  - speaker_id: 音色ID                                          │
│  - encoding: 编码格式 (wav)                                    │
│  - speed_ratio: 语速 (1.0)                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6. save_history - 保存历史

```
┌─────────────────────────────────────────────────────────────────┐
│                    save_history 节点                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 保存对话历史到会话                                       │
│                                                                 │
│  输入:                                                          │
│  - state["text"] (用户输入)                                    │
│  - state["response"] (AI回复)                                  │
│  - state["msg_id"] (消息ID)                                    │
│  - state["user"] (用户名)                                      │
│                                                                 │
│  操作:                                                          │
│  1. history.add_user_message() ──── 添加用户消息              │
│  2. history.add_assistant_message() ──── 添加AI回复           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 7. update_user_memory - 更新用户记忆

```
┌─────────────────────────────────────────────────────────────────┐
│                update_user_memory 节点                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  功能: 更新用户的记忆和画像                                     │
│                                                                 │
│  输入:                                                          │
│  - state["user"] (用户名)                                      │
│  - state["text"] (用户输入)                                    │
│  - state["response"] (AI回复)                                  │
│                                                                 │
│  操作:                                                          │
│  1. user_profile.add_conversation() ──── 添加对话记录         │
│  2. trigger_summary_if_needed() ──── 触发摘要生成             │
│                                                                 │
│  摘要触发条件:                                                  │
│  interaction_count - last_summary_count >= 阈值               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 流式回复模式

系统支持两种回复模式：

### 1. 完整回复模式

```
LLM 完整生成 ──── TTS 完整合成 ──── 一次性返回
```

### 2. 流式同步回复模式

```
┌─────────────────────────────────────────────────────────────────┐
│                    流式同步回复                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  LLM 流式输出: "你｜好｜啊｜，｜今｜天｜天｜气｜真｜好｜！"     │
│                    │                                             │
│                    ▼                                             │
│  句子切分: ["你好啊，", "今天天气真好！"]                       │
│                    │                                             │
│                    ▼                                             │
│  并行 TTS:                                                      │
│  ┌─────────────┐  ┌─────────────┐                              │
│  │ TTS(句子1)  │  │ TTS(句子2)  │                              │
│  └──────┬──────┘  └──────┬──────┘                              │
│         │                │                                      │
│         ▼                ▼                                      │
│  音频1 + 文字1    音频2 + 文字2                                 │
│                    │                                             │
│                    ▼                                             │
│  前端同步播放: 文字和语音同步展示                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
