# 非功能需求

## 性能需求

### 响应时间

| 指标 | 目标值 | 说明 |
|-----|-------|------|
| 弹幕延迟 | < 500ms | 从收到弹幕到广播到前端 |
| AI 回复首字 | < 2s | 从触发回复到输出第一个字 |
| TTS 合成 | < 1s | 单句语音合成时间 |
| 虚拟形象帧率 | >= 30 FPS | 渲染输出帧率 |

### 并发能力

| 指标 | 目标值 | 说明 |
|-----|-------|------|
| WebSocket 连接 | >= 100 | 同时在线连接数 |
| 弹幕处理 | >= 100/s | 每秒处理弹幕数 |
| 点歌请求 | >= 10/s | 每秒处理点歌数 |

### 资源占用

| 指标 | 目标值 | 说明 |
|-----|-------|------|
| CPU 占用 | < 30% | 正常运行时 |
| 内存占用 | < 2GB | 不含模型加载 |
| GPU 显存 | < 4GB | 模型加载后 |
| 磁盘空间 | < 10GB | 含模型文件 |

---

## 可用性需求

### 系统稳定性

| 指标 | 目标值 | 说明 |
|-----|-------|------|
| 系统可用性 | >= 99% | 月度统计 |
| 平均故障恢复时间 | < 5min | 自动恢复 |
| 连续运行时间 | >= 24h | 无需重启 |

### 容错能力

| 场景 | 处理方式 |
|-----|---------|
| B站弹幕服务器断开 | 自动重连，间隔递增 |
| LLM API 超时 | 返回默认回复，记录日志 |
| TTS 合成失败 | 仅返回文字，跳过语音 |
| 数据库连接失败 | 使用内存缓存，记录日志 |

---

## 安全需求

### 数据安全

| 需求 | 说明 |
|-----|------|
| API 密钥保护 | 不在日志中输出，使用环境变量 |
| SESSDATA 保护 | 加密存储，不在前端暴露 |
| 用户数据保护 | 遵守隐私政策，支持数据清除 |

### 访问控制

| 需求 | 说明 |
|-----|------|
| 管理员验证 | 通过 UID 和用户名双重验证 |
| 指令权限 | 非管理员无法执行管理指令 |
| API 限流 | 防止恶意请求 |

---

## 可维护性需求

### 代码质量

| 需求 | 说明 |
|-----|------|
| 代码规范 | 遵循 PEP 8 (Python) / ESLint (TypeScript) |
| 类型提示 | 使用 TypeScript 和 Python Type Hints |
| 文档覆盖 | 核心函数有文档字符串 |
| 单元测试 | 核心模块有单元测试 |

### 日志记录

| 需求 | 说明 |
|-----|------|
| 日志级别 | DEBUG / INFO / WARNING / ERROR |
| 关键操作日志 | 点歌、AI回复、管理员指令 |
| 错误日志 | 包含堆栈信息 |
| 日志轮转 | 按日期轮转，保留 7 天 |
