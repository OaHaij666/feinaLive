# AI 对话 API

## 概述

AI 对话模块提供 AI 主播的智能回复功能，支持流式输出文字和语音，实现文字语音同步播放。

## API 端点

### 流式回复

发送消息获取 AI 主播的流式回复。

```
POST /ai/reply
```

**请求体**：

```json
{
  "user": "用户名",
  "content": "消息内容",
  "uid": 123456
}
```

**请求字段**：

| 字段 | 类型 | 必填 | 说明 |
|-----|------|------|------|
| user | string | 否 | 用户名，默认 "user" |
| content | string | 是 | 消息内容 |
| uid | number | 否 | 用户 UID，默认 0 |

**响应**：

返回 Server-Sent Events (SSE) 格式的流式数据。

```
Content-Type: text/event-stream
Cache-Control: no-cache
Connection: keep-alive
```

**SSE 消息格式**：

```
data: {"type":"start","is_final":false}

data: {"type":"text","text":"你好","is_final":false}

data: {"type":"audio","audio":"base64...","sentence_index":0,"char_offset":0,"char_length":10,"is_final":false}

data: {"type":"end","text":"完整回复文本","is_final":true}
```

---

### SSE 消息类型

#### 1. 开始消息 (start)

标志回复开始。

```json
{
  "type": "start",
  "is_final": false
}
```

---

#### 2. 文字消息 (text)

流式输出的文字片段（按句子）。

```json
{
  "type": "text",
  "text": "这是一个句子。",
  "is_final": false
}
```

---

#### 3. 音频消息 (audio)

对应句子的音频数据。

```json
{
  "type": "audio",
  "audio": "base64编码的音频数据",
  "sentence_index": 0,
  "char_offset": 0,
  "char_length": 20,
  "is_final": false
}
```

**字段说明**：

| 字段 | 类型 | 说明 |
|-----|------|------|
| audio | string | Base64 编码的音频数据 |
| sentence_index | number | 句子索引 |
| char_offset | number | 在完整文本中的字符偏移 |
| char_length | number | 句子字符长度 |

---

#### 4. 结束消息 (end)

标志回复结束。

```json
{
  "type": "end",
  "text": "完整的回复文本",
  "is_final": true
}
```

---

#### 5. 错误消息 (error)

处理过程中发生错误。

```json
{
  "type": "error",
  "text": "错误信息",
  "is_final": false
}
```

---

### 获取 AI 状态

获取 AI 主播当前状态。

```
GET /ai/status
```

**响应示例**：

```json
{
  "is_replying": false,
  "buffer_size": 5,
  "unanswered_count": 2
}
```

**响应字段**：

| 字段 | 类型 | 说明 |
|-----|------|------|
| is_replying | boolean | 是否正在回复中 |
| buffer_size | number | 弹幕缓冲区大小 |
| unanswered_count | number | 未回复的弹幕数量 |

---

### 管理员指令

#### 获取管理员状态

获取管理员指令处理器的状态。

```
GET /ai/admin/state
```

**响应示例**：

```json
{
  "is_sleeping": false,
  "face_mode": "auto",
  "volume": 0.8,
  "is_paused": false,
  "hide_admin_danmaku": false
}
```

**状态字段**：

| 字段 | 类型 | 说明 |
|-----|------|------|
| is_sleeping | boolean | AI 是否休眠 |
| face_mode | string | 表情模式 (auto/happy/sad/angry) |
| volume | number | 音量 (0.0-1.0) |
| is_paused | boolean | 音乐是否暂停 |
| hide_admin_danmaku | boolean | 是否隐藏管理员弹幕 |

---

#### 发送管理员指令

通过 API 发送管理员指令。

```
POST /ai/admin/command
```

**请求体**：

```json
{
  "command": "/sleep"
}
```

**响应示例**：

```json
{
  "success": true,
  "message": "AI主播已进入休眠模式",
  "command": "sleep",
  "state": {
    "is_sleeping": true,
    "face_mode": "auto",
    "volume": 0.8,
    "is_paused": false,
    "hide_admin_danmaku": false
  }
}
```

---

## 管理员指令列表

| 指令 | 说明 |
|-----|------|
| `/sleep` | AI 进入休眠模式，不回复弹幕 |
| `/wake` | AI 唤醒，恢复回复 |
| `/pause` | 暂停音乐播放 |
| `/resume` | 恢复音乐播放 |
| `/next` | 切换下一首歌曲 |
| `/rm` | 移除当前歌曲并禁用 |
| `/hide` | 隐藏管理员弹幕 |
| `/show` | 显示管理员弹幕 |
| `/volume 0.5` | 设置音量 (0.0-1.0) |
| `/face happy` | 设置表情模式 |

## 配置参数

| 参数 | 说明 | 默认值 |
|-----|------|-------|
| host_reply_interval | 回复间隔（秒） | 5 |
| host_max_reply_length | 最大回复长度 | 100 |
| host_model | AI 模型 | doubao-seed-character |
| host_temperature | 温度参数 | 0.7 |
| host_top_p | Top-P 参数 | 0.9 |
| host_max_tokens | 最大 Token 数 | 200 |
