# 播放列表表 (playlist)

## 表概述

播放列表表存储预备歌单，即预先审核过的歌曲列表。点歌时优先从此表匹配，提高点歌成功率和响应速度。

## 表结构

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|-------|---------|------|-------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | - | 自增主键 |
| bvid | VARCHAR(20) | UNIQUE, NOT NULL | - | B站视频BV号 |
| title | VARCHAR(255) | NOT NULL | - | 歌曲标题 |
| artist | VARCHAR(100) | NOT NULL | "" | 歌手/UP主名称 |
| enabled | BOOLEAN | NOT NULL | TRUE | 是否启用 |
| created_at | DATETIME | NOT NULL | NOW() | 创建时间 |

## 字段详解

### bvid

```
类型: VARCHAR(20)
约束: UNIQUE, NOT NULL
说明: B站视频的唯一标识符
示例: BV1xx411c7mD
```

---

### title

```
类型: VARCHAR(255)
约束: NOT NULL
说明: 歌曲标题，用于搜索匹配
```

**注意**：这里的 title 是经过 LLM 验证后的歌曲名，可能与原视频标题不同。

---

### artist

```
类型: VARCHAR(100)
约束: NOT NULL
默认值: ""
说明: 歌手或UP主名称
```

---

### enabled

```
类型: BOOLEAN
约束: NOT NULL
默认值: TRUE
说明: 是否启用该歌曲
```

**用途**：
- `TRUE`：歌曲可被点播和随机选取
- `FALSE`：歌曲被禁用（如用户执行 `/rm` 指令）

---

### created_at

```
类型: DATETIME
约束: NOT NULL
默认值: NOW()
说明: 记录创建时间
```

## 配置示例

```yaml
# config.yaml
default_playlist:
  - bvid: "BV1xx411c7mD"
    title: "歌曲名称"
    artist: "歌手名"
  - bvid: "BV1yy411c7mD"
    title: "另一首歌"
    artist: "另一个歌手"

llm:
  auto_collect_min_views: 20000  # 自动收录的最低播放量
```

## 自动收录逻辑

```
点歌成功
    │
    ▼
获取视频详情
    │
    ▼
播放量 >= auto_collect_min_views?
    │
    ├─ 是 ──▶ 检查 playlist 是否已存在
    │              │
    │              ├─ 不存在 ──▶ 添加到 playlist
    │              │
    │              └─ 已存在 ──▶ 跳过
    │
    └─ 否 ──▶ 不收录
```
