# 项目简介

## 项目背景

飞娜直播间是一个基于AI技术的虚拟主播项目，集成B站弹幕互动、智能回复、音乐播放和虚拟形象渲染等功能。该项目旨在打造一个24小时无人值守的AI直播间，能够与观众进行自然对话、播放背景音乐，并通过虚拟形象与观众互动。

## 核心功能特性

### 🎙️ AI主播对话
基于大语言模型（LLM）的智能对话系统，支持：
- **弹幕智能回复**：自动识别弹幕内容，生成合理的AI回复
- **语音合成（TTS）**：火山引擎声音复刻技术，将文字转换为自然语音
- **记忆系统**：短期记忆（会话历史）、中期记忆（用户画像）、长期记忆（RAG）
- **流式输出**：按句子流式输出文字和音频，实现文字语音同步

### 📺 B站弹幕接入
- WebSocket实时连接B站直播间弹幕服务器
- 支持多种弹幕类型：普通弹幕、高赞弹幕、礼物、欢迎舰长等
- 弹幕拦截与点歌服务

### 🎵 音乐播放系统
- **点歌服务**：支持BV号点歌、歌名点歌、预备歌单点歌
- **LLM验证**：智能识别音乐视频，拒绝非音乐内容
- **自动收录**：高播放量音乐视频自动加入预备歌单
- **播放队列**：最多5首歌曲排队，支持切歌、跳歌

### 🎭 虚拟形象渲染
- 基于EasyVtuber的实时虚拟形象驱动
- 支持THA3/THA4模型
- GPU加速推理（CUDA + ONNX Runtime）
- WebSocket推流到前端展示

### 🖥️ 前端界面
- Vue 3 + TypeScript + Vite
- 实时弹幕展示面板
- 音乐播放器控制
- 虚拟形象显示区域

## 技术选型

### 后端技术栈

| 技术 | 版本 | 用途 |
|-----|------|-----|
| FastAPI | 0.1.0 | Web框架 |
| SQLAlchemy | - | ORM数据库访问 |
| asyncio | 内置 | 异步编程 |
| LangGraph | - | AI对话流程编排 |
| ONNX Runtime | - | AI模型推理加速 |
| bilibili-api | - | B站API封装 |

### 前端技术栈

| 技术 | 版本 | 用途 |
|-----|------|-----|
| Vue 3 | 3.x | UI框架 |
| TypeScript | 5.x | 类型系统 |
| Vite | 5.x | 构建工具 |
| Pinia | - | 状态管理 |
| Tailwind CSS | 3.x | 样式框架 |

### 基础设施

| 组件 | 用途 |
|-----|------|
| nginx-rtmp | RTMP推流服务器 |
| SQLite | 轻量级数据库 |
| CUDA | GPU加速 |

## 适用场景

1. **虚拟主播运营**：24小时AI直播间，降低运营成本
2. **弹幕互动研究**：弹幕处理、AI对话实验平台
3. **虚拟形象演示**：实时驱动虚拟形象的各种应用
