# 点歌业务流程

## 概述

点歌服务允许观众通过发送弹幕来请求播放音乐，系统会自动解析请求、搜索歌曲、验证内容并加入播放队列。

```
│         │              │                                        │

```

## 错误处理

| 错误类型 | 错误信息 | 原因 |
|---------|---------|------|
| 用户限制 | "你已有N首歌在队列中" | 用户点歌数超限 |
| 队列限制 | "播放队列已满" | 队列已满 |
| 无效请求 | "未找到有效的BV号或歌名" | 无法解析 |
| 搜索失败 | "未找到匹配的歌曲" | 歌曲不存在 |
| 验证失败 | "该视频不是音乐类型" | LLM判定非音乐 |
| 获取失败 | "获取音频URL失败" | B站API错误 |
