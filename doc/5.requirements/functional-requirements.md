# 功能需求

## 需求概述

飞娜直播间是一个基于 AI 技术的虚拟主播系统，旨在实现 24 小时无人值守的智能直播。

## 功能模块

### 1. B站弹幕接入模块

| 需求ID | 需求名称 | 优先级 | 描述 |
|-------|---------|-------|------|
| F1.1 | 实时弹幕接收 | P0 | 通过 WebSocket 实时接收 B站直播间弹幕 |
| F1.2 | 弹幕类型支持 | P1 | 支持普通弹幕、礼物消息、进入消息等 |
| F1.3 | SESSDATA 配置 | P1 | 支持配置 SESSDATA 进行身份验证 |
| F1.4 | 多直播间支持 | P2 | 支持同时连接多个直播间 |
| F1.5 | 断线重连 | P1 | 自动重连机制，保证连接稳定 |

### 2. AI 对话模块

| 需求ID | 需求名称 | 优先级 | 描述 |

## 管理员指令列表

| 指令 | 功能 | 示例 |
|-----|------|------|
| /sleep | AI 进入休眠模式 | /sleep |
| /wake | AI 唤醒 | /wake |
| /pause | 暂停音乐播放 | /pause |
| /resume | 恢复音乐播放 | /resume |
| /next | 切换下一首歌曲 | /next |
| /rm | 移除当前歌曲并禁用 | /rm |
| /volume N | 设置音量 (0-1) | /volume 0.5 |
| /hide | 隐藏管理员弹幕 | /hide |
| /show | 显示管理员弹幕 | /show |
| /face MODE | 设置表情模式 | /face happy |

## 用户故事

### US-001: 观众点歌

```
作为 观众
我想要 通过弹幕点歌
以便于 在直播间听到我想听的音乐

验收标准:
- 发送 "点歌 BV1xx..." 可添加歌曲
- 发送 "点歌 歌曲名" 可搜索添加
- 点歌成功后收到通知
- 点歌失败时收到错误提示
```

### US-002: 观众与 AI 对话

```
作为 观众
我想要 与 AI 主播对话
以便于 获得互动体验

验收标准:
- 发送弹幕后 AI 会回复
- AI 能记住我的历史互动
- 我可以通过 /clear 清除记忆
```

### US-003: 管理员控制直播

```
作为 管理员
我想要 通过弹幕控制直播
以便于 管理直播间状态

验收标准:
- 管理员指令能正确执行
- 非管理员无法执行管理指令
- 状态变更会广播通知
```
