# UP主视频表 (up_videos)

## 表概述

UP主视频表存储信任 UP主发布的视频信息，用于点歌时快速搜索匹配，避免每次都调用 B站 API。

## 表结构

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|-------|---------|------|-------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | - | 自增主键 |
| bvid | VARCHAR(20) | UNIQUE, NOT NULL | - | B站视频BV号 |
| title | VARCHAR(255) | NOT NULL | - | 视频标题 |
| up_name | VARCHAR(100) | NOT NULL | - | UP主名称 |
| up_uid | BIGINT | INDEX | - | UP主UID |
| duration | INTEGER | NOT NULL | 0 | 视频时长（秒） |
| cover_url | VARCHAR(500) | NOT NULL | "" | 视频封面URL |
| fetched_at | DATETIME | NOT NULL | NOW() | 抓取时间 |

## 字段详解

### bvid

```
类型: VARCHAR(20)
约束: UNIQUE, NOT NULL
说明: B站视频的唯一标识符，格式为 BV + 10位字符
示例: BV1xx411c7mD
```

**BV号格式**：
- 以 "BV" 开头
- 后跟 10 位字母数字组合
- 总长度固定 12 位

---

### title

```
类型: VARCHAR(255)
约束: NOT NULL
说明: 视频标题，用于搜索匹配和显示
```

---

### up_name

```
类型: VARCHAR(100)
约束: NOT NULL
说明: UP主昵称
```

---

### up_uid

```
类型: BIGINT
约束: INDEX
说明: UP主的B站UID，用于按UP主筛选视频
```

**索引用途**：快速查询某UP主的所有视频

---

### duration

```
类型: INTEGER
约束: NOT NULL
默认值: 0
说明: 视频时长，单位为秒
```

---

### cover_url

```
类型: VARCHAR(500)
约束: NOT NULL
默认值: ""
说明: 视频封面图片URL
```

---

### fetched_at

```
类型: DATETIME
约束: NOT NULL
默认值: NOW()
说明: 视频信息抓取时间，用于增量更新判断
```

## 配置示例

```yaml
# config.yaml
trusted_ups:
  - uid: 123456789
    name: "音乐UP主A"
  - uid: 987654321
    name: "音乐UP主B"

up_videos:
  incremental_days: 5    # 增量刷新间隔
  full_refresh_days: 30  # 全量刷新间隔
```
